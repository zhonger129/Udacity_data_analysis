Data cleaning¶

Being observed from above section, there were only 714 out of 891 passengers have age information, since I need to have analysis for age and survivors, I have to handling the missing values for age. we assume that missing values are distributed similarly to the values that are present, here, I will fill the missing age with mean.

Next, as the age are discrete with too many values, to compare the age and survivor and clearly visualize the ages with plot for my further analysis, I am going to group the age by every ten years old for passengers.

In [10]:
# 1. Replace the missing age with mean
passenger_mean_age=titanic_df['Age'].mean()
titanic_df['Age'].fillna(passenger_mean_age,inplace=True)

# 2. Group passenger's age as following

# < 10    : less_than_ten
# 10 - 20 : ten_to_twenty
# 20 - 30 : twenty_to_thirty
# 30 - 40 : thirty_to_fourty
# 40 - 50 : fourty_to_fifty
# 50 - 60 : fifty_to_sixty
# 60 - 70 : sixty_to_seventy
# > 70    : greater_than_seventy

def add_passenger_Age_stage(age):
    if age < 10:
        return 'less_than_ten'
    elif age < 20:
        return 'ten_to_twenty'
    elif age < 30:
        return 'twenty_to_thirty'
    elif age < 40:
        return 'thirty_to_fourty'
    elif age < 50:
        return 'fourty_to_fifty'
    elif age < 60:
        return 'fifty_to_sixty'
    elif age < 70:
        return 'sixty_to_seventy'
    else:
        return 'greater_than_seventy'

# Add new columns to existing dataframe
def patch_df(titanic_df):
    titanic_df['Age_stage'] = titanic_df['Age'].apply(add_passenger_Age_stage)

# new dataframe to store passenger survived 
def survived_passenger_df(titanic_df):
    return titanic_df[titanic_df.Survived==1]

# Calculate the number of group data
def count_group_data(titanic_df,group_key):
    return titanic_df.groupby(group_key)['PassengerId'].count()

# Calculate the rate of survived passenger
def rate_of_survived_passenger(group_data,group_key):
    # Calculate number of all passenger in the group 
    all_passenger_of_each_group = count_group_data(titanic_df,group_key)

    # Calculate number of all passenger survived in the group 
    survived_passenger_of_each_group = count_group_data(survived_passenger_df(group_data),group_key)
    
    # Return the rate of survived passenger
    return survived_passenger_of_each_group / all_passenger_of_each_group

# Print the histogram chart 
def print_bar(group_data,title,xlab,ylab):
    sns.set_style("whitegrid")
    group_data.plot.bar(title=title,
                        figsize=(5,5),
                        color='b',
                        alpha=0.6
                       )
    xlabel(xlab,fontsize=12 )
    ylabel(ylab,fontsize=12 )

    plt.title(titel)

Pearson correlation r¶

# This function is to calculate Pearson's r
# r = average of (x in std unit) * (y in std unit)
def correlation(x,y):
    std_x = (x - x.mean()) / x.std(ddof=0)
    std_y = (y - y.mean()) / y.std(ddof=0)
    
    return (std_x * std_y).mean()

Conclusions¶

Summy¶

  1. There were 342 survivors out of 891 passenger.
  2. Passenger hold first ticket class had the most survivors
  3. Passenger embarked from Cherbourg had the most survivors
  4. Female had a higher count for survivors
  5. There were no survivors for age group [70,80). observed from each age group, age falled in [0,10) had the highest rate of passengers survived

Limitations of analysis¶

There is no statistical analysis to support these results, so it is not known whether the results here are caused by real differences or noise.

In addition, the age field has some missing values, and we are replacing the missing value with the average age of all passengers. Such a practice would make most passenger fall in the age group and affect the results of the comparison.
